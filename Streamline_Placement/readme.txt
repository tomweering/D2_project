----------------------------------------
STREAMLINE PLACEMENT MODULE INSTALLATION
----------------------------------------
List of modules required:
(NB. if using virtual environment, make sure to install these modules in the environment)
- SciPy
- Numpy
- Matplotlib
- VTK (must be version later than 9.1.0 - this latest version is not in conda directory so must be installed with pip)
- Pyvista (cannot be found by conda, must be installed using pip)
